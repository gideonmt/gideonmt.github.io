Software

  • Operating System: I use Arch Linux becuase of its many customization options and reliability. Setting it up might take a little longer, but it provides everything I need and works great.

  • Window Manager: For my WM I use Hyprland. It's a great tiling window manager.

  • Browser: I use Firefox with a custom user.js file to strenghten privacy and remove unnecessary features. I also use the following extensions:

    • uBlock Origin: For blocking ads and malicious sites.
    • Simple Login: To quickly create proxy emails.
    • LibreRedirect: Redirects links from YouTube, Twitter, TikTok, and Reddit to privacy-focused front-end alternatives.

  • Terminal: I use Kitty with the JetBrains Mono font and the same colors as the VSCodium Dark Modern theme.

  • Shell: For my shell Zsh works great. I use zsh4humans as the configuration.

  • Code Editor: VSCodium is a telemetry-free version of Visual Studio Code. I also use GitHub Copilot and several other extensions to enhance my coding experience. I use the JetBrains Mono font, and the Dark Modern theme.
